# GoBang 蓝牙五子棋，了解一下，基于Socket实现的
## 整体功能
  本游戏主要含有四个功能设计：AI算法设计、人人对战、蓝牙联机、存档，用户通过点击相应按钮进入相应界面。在棋盘界面，用户可以查看棋局进行时间，可以悔棋（注：人机对战没有悔棋），可以存档，右上角出现的黑白棋子提醒用户哪方执子，白棋表示现在白方执子，反之亦然，当棋局有一方获得胜利后结束游戏，系统会提示是否继续游戏或者返回上一界面
